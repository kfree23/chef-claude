# Chef Claude

An AI-powered recipe generator — input the ingredients you have on hand and get a custom recipe suggestion. Built with React and the Anthropic API.

## How It Works
- Add ingredients
- Click to generate a recipe
- Chef Claude suggests a recipe using your ingredients (plus a few extras if needed)
- Recipe is formatted in clean, readable markdown

## Features
- Dynamic ingredient list management
- AI-generated recipes via Claude API
- Markdown rendering for easy-to-read recipes
- Simple, intuitive UI

## Tech Stack
- React
- JavaScript
- Anthropic API (Claude)
- react-markdown
